## 8. Waves

A function satisfies the wave equation if it has the form:

y(x,t) = f(x ± vt)

### Check each option:

a) y(x,t) = A cos(kx² − ωt)  
Not in the form f(x ± vt) → ❌ Not a traveling wave  

b) y(x,t) = A(x − vt)²  
Matches f(x − vt) → ✅ Traveling wave  

c) y(x,t) = A log(x + vt)  
Matches f(x + vt) → ✅ Traveling wave  

### Final Answer:
**b) and c)**
