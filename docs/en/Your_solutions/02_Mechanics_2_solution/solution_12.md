# Work and Energy Analysis (Constant Force)

This report details the motion of a 2 kg mass under a constant force field and verifies the Work-Energy Theorem.

## 1. Kinematic Equations
* **Acceleration:** $\vec{a} = [3, 1] \text{ m/s}^2$
* **Velocity:** $\vec{v}(t) = [3t + 1, t - 1] \text{ m/s}$
* **Position:** $\vec{r}(t) = [1.5t^2 + t, 0.5t^2 - t] \text{ m}$

## 2. Trajectory Description
The trajectory is a **parabola** in the 2D plane. Because the acceleration components are constant, the path follows a quadratic relationship between $x$ and $y$.

## 3. Work-Energy Verification at $t = 3s$

| Parameter | Value |
| :--- | :--- |
| Displacement $\Delta\vec{r}$ | $[16.5, 1.5]$ m |
| Work Done ($W$) | **102 J** |
| Initial $KE$ | 2 J |
| Final $KE$ | 104 J |
| Change in $KE$ | **102 J** |

**Conclusion:** The work done by the force exactly matches the change in kinetic energy, confirming the **Work-Energy Theorem**.
